BendersLib Documentation
===================================

This is a modern implementation (``branch-and-check``) for Benders Decomposition in Python. It includes classical Benders decomposition and its variants (``Generalized Benders``, ``Combinatorial Benders``, ``Logic-based Benders``, etc.), enhanced by a series of accelerating techniques (``Pareto-optimal cut``, ``local branching``, etc.).
